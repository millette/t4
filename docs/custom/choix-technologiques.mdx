# Choix technologiques

## Architecture

- Postgres / GraphQL / [Hasura](https://hasura.io/)
- NodeJS / React / [Next.js](https://nextjs.org/)
- Micro-services (séparer les API du front-end)
- S3 compatible storage [MinIO](https://min.io/) pour fichiers, images, pdf...

### Minio

**Détails:** [minio](/custom/minio)

Pour les fichiers attachés, les vidéo, images, bref, tout ce qui ne va pas dans la DB.

On peut utiliser `nginx` pour _resizer_ les images et les conserver dans un cache en proxy à `MinIO`.
